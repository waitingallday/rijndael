Simple AES (Rijndael) codec for obfuscating links badly.

What is this even for

This trick should be enough to fool any basic scraping tech, by encoding a given string via an AES cipher.

Is that the private key included in plain text

A human who looks at the included codec will note the private key is, in fact, included in plain text. This is just a quick, trivial way of obscuring links, especially for email.
